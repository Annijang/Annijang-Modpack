######################################
#          itemframes.json          #
######################################
  该文件里包含物品展示框在生成时自带物品时的随机信息。
物品展示框会生成在某些特殊的房间里和走廊里。
关于物品随机信息的更多信息请查看本文件的最底部。
######################################
#         ItemRandomizers           #
######################################
描述物品和物品被选中的概率。
 - entries: 一个对象，每个对象都对应一个值，该值就是物品被选中的概率。
      所有的概率总和不能超过1.0！
 - defaultItem: 如果没有选中“entries”里的对象则会使用该默认物品来作为补充。
      例如：“entries”对象里的所有概率总和加起来只有0.6（60%），那么就会有0.4（40%）的概率会选择“defaultItem”里的物品来作为补充，默认都是air（空气）。

下面是一个例子：
{
  "entries": {
    "minecraft:cobblestone": 0.25,
    "minecraft:air": 0.2,
    "minecraft:stone_sword": 0.1
  },
  "defaultItem": "minecraft:iron_axe"
}
默认物品是“minecraft:iron_axe”（铁斧），在选择物品作为盔甲架/展示框里显示时有25%的概率选择圆石、20%的概率选择空气、10%的概率选择为石剑，那么选择默认物品（铁斧）的概率是：100%-25%-20%-10%=45%，也就是说如果未选中圆石、空气、盾牌时，有45%的概率会默认选择铁斧来作为备用的补充物品。
