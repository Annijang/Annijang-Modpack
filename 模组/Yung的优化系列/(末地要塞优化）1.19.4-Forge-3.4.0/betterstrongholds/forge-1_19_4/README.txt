######################################
#             ores.json              #
######################################

  该文件里有个“方块选择”的功能，设置指定矿物被选中的概率。
矿物的概率主要在不同的结构里的特殊房间生效，并且会有概率生成扎堆的矿物。
关于方块选择的更多信息，请查看本文件的底部。

######################################
#          rareblocks.json           #
######################################

  该文件里有个“方块选择”的功能，设置指定方块被选中的概率。
方块的概率主要在大型图书馆里生效，其中会有两个特殊的方块生成。
关于方块选择的更多信息，请查看本文件的底部。

######################################
#          armorstands.json          #
######################################

  该文件里包含盔甲架在生成时自带盔甲和盔甲部位的物品随机信息。
普通的盔甲架会生成在锻造室里，而稀有的盔甲架只会生成在BOSS房间里。
关于物品随机信息的更多信息请查看本文件的最底部。

######################################
#          itemframes.json          #
######################################

  该文件里包含物品展示框在生成时自带物品时的随机信息。
物品展示框会生成在锻造室和储藏室里。
关于物品随机信息的更多信息请查看本文件的最底部。

######################################
#         BlockSetSelectors          #
######################################

描述方块和方块被选中的概率。
 - entries: 一个对象，每个对象都对应一个值，该值就是方块被选中的概率。
      所有的概率总和不能超过1.0！
 - defaultBlock: 如果没有选中“entries”里的对象则会使用该默认方块来作为补充。
      例如：“entries”对象里的所有概率总和加起来只有0.6（60%），那么就会有0.4（40%）的概率会选择“defaultItem”里的物品来作为补充。

下面是一个例子：
"entries": {
  "minecraft:cobblestone": 0.25,
  "minecraft:air": 0.2,
  "minecraft:stone_bricks": 0.1
},
"defaultBlock": "minecraft:oak_planks"

默认方块是“minecraft:oak_planks”（橡木木板），在选择指定方块作为矿物位置时有25%的概率选择圆石、20%的概率选择空气、10%的概率选择为石砖，那么选择默认方块（橡木木板）的概率是：100%-25%-20%-10%=45%，也就是说如果未选中圆石、空气、石砖时，有45%的概率会默认选择橡木木板来作为备用的补充方块。

######################################
#         ItemSetSelectors           #
######################################

描述物品和物品被选中的概率。
工作原理与“BlockSetSelectors”一致，只是将方块方块替换为物品了而已。
